项目#

pinyin#

最流行的开源汉字转拼音库之一，使用 TypeScript/JavaScript 实现，运行速度快，字库全，支持简体中文、繁体中文和各种偏僻汉字。

  • WebSite: https://pinyin.js.org
  • GitHub: https://github.com/hotoo/pinyin

Markline#

基于 Markdown 格式的轻量级时间线(Timeline)软件，目前有 Web 版、Obsidian 版，其他版本（如 VSCode、Eleteron）开发中， 敬请期待。如果您有其他需求，请提交 Issue、或给我发送邮件。

  • WebSite: https://hotoo.github.io/markline
  • GitHub: https://github.com/hotoo/markline
  • Obsidian: https://github.com/hotoo/obsidian-markline

演示：

  • 我的一生： https://hotoo.github.io/life/
  • 曾养过的 Samoyed 犬： https://hotoo.github.io/samoyed/ 名字叫 Summer，因为是夏天出生。

gitopen#

命令行(CLI)用户的福音，直接在命令行环境一键提交 PR/MR、打开 Git 仓库主页、Issues 页面等。

  • GitHub: https://github.com/hotoo/gitopen

git-branch-alias#

一个给 Git 分支添加别名的命令行工具，解决蚂蚁集团、字节跳动、抖音集团等公司级 Git 平台的分支命名规范复杂不易识别、不易输入的问题。

  • GitHub: https://github.com/hotoo/git-branch-alias

GitLab-Notifications#

早年开源的 GitLab 没有通知中心，通知都是通过邮件发送和接收。项目协同时会很麻烦，需要打开邮件通知（这样每封邮件都会打扰你）， 收到提及或关注的通知时，需要先打开邮箱、再点击链接到相关的 Issue/MergeRequest，处理到什么位置和进度都是不自动化的。

这个浏览器插件可以关注特定仓库，或者被提及的任务才给你通知，避免打扰。同时可以点击通知消息一键打开相关 Issue/MergeRequest， 直接定位到上次处理的位置。通知中心还会列出所有待处理的任务，提高异步处理任务效率。

  • GitHub: https://github.com/hotoo/gitlab-notifications
  • GitLab: https://gitlab.com/hotoo/gitlab-notifications

rm-trash#

MacOS 系统中命令行(CLI)环境下安全的一键删除文件、删除目录的软件。发现误删时，可以在回收站中一键放回原处。

  • GitHub: https://github.com/hotoo/rm-trash

vim desktop calendar#

一个 Vim 主题、可一键打印、方便放置在办公桌面上的纸质日历。

继承 Bram Moolenaar (米勒) 的遗志。

  • WebSite: link
  • GitHub: https://github.com/hotoo/vim-desktop-calendar

pangu.vim#

盘古开天地，化混沌为清朗，给汉字和英文之间添加盘古之白的 Vim 插件，在保存文件时自动规范化。

  • GitHub: https://github.com/hotoo/pangu.vim

canendar-vim#

由于 mattn/calendar-vim 长期未维护，提交的年报、月报、周报等功能未被合并，因此保持了 Fork。

  • GitHub: https://github.com/hotoo/calendar-vim

jsgf.vim#

增强的 TypeScript, JavaScript, React, Vue, package.json 版 Vim gf 指令，一键打开跳转到光标所在文件。

  • GitHub: https://github.com/hotoo/jsgf.vim
  • For Alibaba's Eggjs Framework: https://github.com/eggjs-gf.vim
  • For ByteDance's Gulu Framework: https://github.com/hotoo/gulu-gf.vim

UniString#

一个高性能的使 TypeScript/JavaScript 支持 Unicode String 的类库，让 3 字节、4 字节或更长字节的字符、Emoji 等作为完整的字符进行处理，避免将多字节字符错误的拆断。

  • GitHub: https://github.com/hotoo/unistring

detector#

一个浏览器环境识别库，早年很受欢迎，甚至得到 webpack 的青睐，奈何我当时认知不够，拒绝了 webpack 的橄榄枝。

  • GitHub: https://github.com/hotoo/detector

orderby#

一个类似 SQL order by 排序语法的函数库，让排序操作快速、清晰、易读。

  • GitHub: https://github.com/hotoo/orderby
