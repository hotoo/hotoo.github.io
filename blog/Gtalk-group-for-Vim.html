欢迎加入 Vim Talks™(vimcn@chatterous.com) 这个 Gtalk 群，任何人可以随意加入， 没有任何限制。由于非常可惜的建废了一个名为 Vim 迷 (Vimi) 的群，只好用了这个 大言不惭的名称。

白色背景 透明背景

另外向对 Vim 感兴趣的朋友推荐一些 有用或有趣的资源。

警告： vimcn 是通过 chatterous.com 实现的一个 Google Talk 群，目前处于公开/开放的状态， 任何人可以自由的加入或退出，包括聊天内容在内，没有任何审查机制。 甚至通过 http://www.chatterous.com/vimcn/ 匿名用户也可以跟群友聊天。 而 http://www.chatterous.com/vimcn/ 是一个网页，所以也会被搜索引擎索引。

如果你觉得这个损害了你的隐私，或者对你构成威胁，请慎重加入。

为此，我们也创建了一个 投票，以此来决定群的公开程度。

有多种加入方法：

  1. 直接访问 http://chatterous.com/vimcn/ GFW 可以在线匿名或具名聊天， 与传统的聊天室相似。
  2. 在 Gtalk 里加 vimcn@chatterous.com 为好友，并向其发送内容为 @join 的消息。
  3. 向 vimcn@chatterous.com 发送内容为 @join 的邮件。
  4. 或用手机向 41411 发送内容为 CHTR JOIN vimcn 的短信（因为发不起国际短信， 这个我没有尝试）

注意：由于 chatterous.com 目前仅支持 Jabber 的开放 IM 协议，所以推荐 使用 Gtalk 或其他支持 Jabber 协议的帐号及软件（如 Gtalk, Pidgin）。

使用帮助(针对Gtalk)：

  • @join 加入群。
  • @help 查看所有可用的命令。
  • @stop 暂停接收群消息，时效 1 天。
  • @go 开始接收群消息。
  • @members 查看群成员， 注意：这是个复数，记得加 s ，很多朋友都输入错误，这是个讨厌的命令，又不能看实时在线人数。
  • @get # 查看倒数第 # 条之后的消息（默认是 1，最大为 100）。
  • @snooze # 暂停 # 分钟不接收群消息，（默认 20 分钟）。
  • @leave 退群。

更多内容参考 GTalk 及其群聊机器人支持 Chatterous

技巧

群名称
为了便于大家找到群相关的讯息，群名称加上了这个网页的地址（因为 chatterous.com 的描述信息不显示到群 Bio 才出此下策），建议在添加机器人之后修改群（机器人）名称为 “Vim Talks™”（或任何你喜欢名称，例如群友提的“Vim 闲话”，“为爱而摸”），避免因群名称过长而受到干扰。

群 LOGO
将上面（或者任何你喜欢）的图片保存到本地，到 Gmail 的联系人(Contacts)、或者你的客户端里修改群机器人的头像即可。

修改昵称
如果希望向群友显示自己的昵称，你需要在 chatterous.comGFW 上注册，并关联到自己的 GTalk 账户。 详细步骤参考 NIX 的分享

缺陷 & 不足

不能像 QQ 一样方便的发送图片并直接预览。GTalk 已经支持发送文件，而且在新版的 Pidgin 中发送图片也支持直接预览，这跟平台相关。如果确实需要分享图片/截图，可以使用第三方的图片服务（如 Flickr，Dropbox 的Public目录，Yupoo 等），将图片所在的链接发送到群里即可。

没有群空间，用来存储/分享文件。当然你同样可以使用第三方存储服务（Dropbox，Windows Live Sync ^SkyDrive^，Google Docs 支持上传任意类型的文件，RapidShare 等）来实现。

Gtalk 群的使用：

鉴于 Gtalk 的特殊性，大段代码最好不要作为聊天内容直接发送，推荐的方法是， 将代码贴到专业的第三方网站（如 Snipt, PasteBay,CopyTaste,Ubuntu Paste,Pastebin），然后将网址贴到聊天消息里。这有个好处是， 避免大段内容影响阅读，专业的第三方服务能保持代码原文不被转义，支持代码着色， 甚至可以直接运行并得到结果，如果是第三方服务的会员，也可以方便管理自己的代码。

有些 *nix 高手偶尔会频繁的尝试一些高级命令，例如 "@online", "@ls" 之类。我个人希望群聊是本着分享和讨论交流的宗旨进行，当然闲聊也可以，话唠也行，但是垃圾信息不要太多以至影响大家正常的交流。

为什么不是 QQ 群？

米兰达警告(Miranda Warning)：“你有权使用 QQ，但是你在 QQ 上所说的一切将被作为呈堂证供。Do you understand?”

GTalk 没有人数限制，不需要冲Q币来增加某些服务，而受到更多不可理喻的限制；可以轻易的加入或退出群，或随时暂停接收他人的讨论；具有完整的开放性，有非常多的官方或第三方客户端可以用，包括Windows,Linux,Mac平台；更尊重 言论自由，没有无理取闹的审查制度，不会因为胡萝卜、李子和温习功课而被敏感词。

另外 TM2009 在 Windows 7 下导致未明原因的频繁假死问题。
