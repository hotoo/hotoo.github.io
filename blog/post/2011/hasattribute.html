深入 hasAttribute#

众所周知，IE 的 HTMLElement 对象不支持 hasAttribute 方法，于是有了如下实现： 它针对不支持 hasAttribute 方法的浏览器，检查 getAttribute 得到的是否等于 null， 如果是 null 则表明这个 HTMLElement 不存在指定的 Attribute。很聪明的一个方法。

1function hasAttr(elem, attr){
2    if(elem.hasAttribute){
3        return elem.hasAttribute(attr);
4    }
5    return null!=elem.getAttribute(attr);
6}

最近写了个原理简单的基于 Select+Input 的可编辑 Combox 控件，用到了这个方法来检查目标 select 对象是否存在如 multiple, editable 这样的属性，来实现不同的 Combox 版本。其他 浏览器上跑的很好，但是在 IE6 上却总是被初始化为 multiple 的实例，很明显，这个 hasAttribute 的实现出问题了。

原来针对 IE6，还有稍多一点的特殊性：

  • input.getAttribute("checked") 返回 Boolean 值，true/false；
    • input[type=checkbox|radio].getAttribute("checked") 根据他的 checked 状态相关；
    • input[type=others...].getAttribute("checked") 始终返回 false；
  • select.getAttribute("multiple") 返回 Boolean 值；
  • select>option.getAttribute("selected") 返回 Boolean 值。

虽然可以通过 getAttribute("checked") 判断 checkbox/radio 的属性状况，但对于 其他类型的 input 元素，是无法准确获得 checked 属性的。

下面是比较完整一点的实现：

1function hasAttr(elem, attr){
2    var tag = elem.tagName.toLowerCase();
3    attr = attr.toLowerCase();
4    if(elem.hasAttribute){
5        return elem.hasAttribute(attr);
6    }
7    if(navigator.userAgent.indexOf("MSIE 6")>0 &&
8        (("input"==tag && "checked"==attr) ||
9        ("option"==tag && "selected"==attr) ||
10        ("select"==tag && "multiple"==attr))){
11        return elem.getAttribute(attr);
12    }
13    return null!=elem.getAttribute(attr);
14}

Update 2011/07/26#

继续发现新坑：IE 中下面的代码

1<p></p>
2<q onclick="alert(0);"></q>
3
4<script>
5    p.onclick = function(){};
6
7    alert(p.getAttribute("onclick"));
8    alert(q.getAttribute("onclick"));
9</script>

输出：

1function(){}
2
3function onclick()
4{
5alert(0);
6}

所以上面的 hasAttr 函数是无法正确返回这种在脚本中设置 onclick 值的。

对策：

1var hasAttr = function(elem, attr){
2    if(!elem || 1!=elem.nodeType){return false;}
3    if(elem.hasAttribute){return elem.hasAttribute(attr);}
4    // for IE, not perfect.
5    // @see http://www.patmullin.com/weblog/2006/04/06/getattributestyle-setattributestyle-ie-dont-mix/
6    attr = attr.toLowerCase();
7    if("style" == attr){return "" !== elem.style.cssText;}
8    if(navigator.userAgent.indexOf("MSIE 6")>0 &&
9        (("input"==tag && "checked"==attr) ||
10        ("option"==tag && "selected"==attr) ||
11        ("select"==tag && "multiple"==attr))){
12        return elem.getAttribute(attr);
13    }
14    var val = elem.getAttribute(attr);
15    if(null == val){return false;}
16    else if("function" == typeof(val)){
17        return val.toString().indexOf("function "+attr+"()") == 0;
18    }else{return true;}
19};

Update 2011/10/26#
